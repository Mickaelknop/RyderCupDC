La synchronisation cloud est intégrée à l'application : tous les appareils ouvrant ce fichier (ou son lien) voient les scores en temps réel, automatiquement.

Partage ce fichier (ou héberge-le) à tous les joueurs. Les scoreurs vont dans « Saisie », les autres regardent « Live ».

⚠️ La saisie ne se synchronise pas ?

Si le bandeau rouge « écriture refusée » apparaît, c'est que ta clé d'accès JSONBin n'a que la permission de lecture. Les permissions d'une clé ne sont pas modifiables — il faut en créer une nouvelle :

1. Sur jsonbin.io → API Keys → Access Keys
2. Crée une nouvelle clé en cochant READ ET UPDATE (et idéalement CREATE)
3. Copie la nouvelle clé, puis remplace dans le fichier HTML la valeur de accessKey (tout en haut, section §0 CONFIG)
4. Recharge la page sur tous les appareils
